# STEAM Project Workspace

This repository serves as a central hub for STEAM (Science, Technology, Engineering, Arts, Mathematics) education resources, project documentation, and collaborative materials.

## Repository Structure

- `docs/` — Project documentation, slides, and shared resources
- `profile/` — Team or project profile information

## Related Repositories

- **Digital Proficiency Kit**  
  [github.com/STEAM-C3T/digital-proficiency-kit](https://github.com/STEAM-C3T/digital-proficiency-kit)  
  Browser-based learning modules for HTML, CSS, and JavaScript in STEAM education.

- **Teacher Toolkit**  
  [github.com/STEAM-C3T/teacher-toolkit](https://github.com/STEAM-C3T/teacher-toolkit)  
  Lesson plans, rubrics, and professional development resources for teachers using the Digital Proficiency Kit.

- **Learning Materials Repository**  
  [github.com/STEAM-C3T/dpk-learning-materials](https://github.com/STEAM-C3T/dpk-learning-materials)  
  Classroom-ready learning materials for each module and unit of the Digital Proficiency Kit. Includes:

  - Marp slide decks (presentations)
  - Step-by-step tutorials
  - Student and teacher workbooks
  - Templates for new modules/units

  > **Purpose:**
  > The Learning Materials repository provides accessible, classroom-ready resources for teachers and students. It is designed to support novice and experienced educators in delivering web literacy and digital skills in STEAM contexts. All materials are open-source and mapped to the Digital Proficiency Kit modules.

## How to Use

- Browse the `docs/` folder for project-wide resources.
- See the linked repositories above for curriculum, teaching tools, and ready-to-use classroom materials.
